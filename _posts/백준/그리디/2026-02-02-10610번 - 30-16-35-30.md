3의 배수와 9의 배수를 자릿수를 더해서 나온 값이 3이나 9로 나누어 떨어지면 그 자릿수를 조합하여 나오는 수는 모두 3 또는 9의 배수이다. 그리고 해당 문제에서는 30, 즉 3 x 10의 배수이므로 10의 배수의 조건과 합치면 된다. 바로 자릿수에 0이 포함되어야 한다는 것.

그래서 그냥 조건을 만족하는지 확인만 하고 만족한다면 내림차순 정렬해버리면 된다. 알고 보면 너무 간단한 문제였다. 

수학은 역시나 어려워...

```java
import java.io.*;  
import java.util.Arrays;  
  
public class Main {  
    static BufferedReader br = new BufferedReader(new InputStreamReader(System.in));  
    static BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));  
    static StringBuilder sb = new StringBuilder();  
  
    public static void main(String[] args) throws IOException {  
        String str = br.readLine();  
  
        int numberSum = 0;  
        boolean zeroFlag = false;  
  
        for (int i = 0; i < str.length(); i++) {  
            int curr = str.charAt(i) - '0';  
            if (curr == 0) {  
                zeroFlag = true;  
            }  
            numberSum += curr;  
        }  
  
        if (!(numberSum % 3 == 0) || !zeroFlag) {  
            sb.append(-1);  
        } else {  
            char[] strArray = str.toCharArray();  
            Arrays.sort(strArray);  
  
            for (int i = strArray.length - 1; i >= 0; i--) {  
                sb.append(strArray[i]);  
            }  
        }  
  
        bw.write(sb.toString());  
        bw.flush();  
        bw.close();  
        br.close();  
    }  
}
```
