여러 로프를 사용해서 들어올릴 수 있는 최대 중량을 구하면 된다. 예제를 보면 각 15, 10키로를 버틸 수 있는 로프가 있다. 15키로 로프를 하나를 썼을 때는 최대 중량은 15이다. 만약 10키로 중량인 로프를 같이 쓴다면, 15키로 중량 로프도 들게 되는 중량이 10키로가 되기 때문에 최대 중량은 20이 된다.

뭐... 중량이 각 15, 10이기 때문에 총 중량은 25이고, 이걸 로프의 개수인 2로 나누면 각 로프에 가해지는 중량은 12.5일테지만, 10키로 중량 로프는 이걸 버티지 못하기 때문에 10키로 로프에 맞춰야 한다는 뜻.

총 시간 복잡도는 O(NlogN + 2N) = O(NlogN)이 된다.

```java
import java.io.*;  
import java.util.Arrays;  
  
public class Main {  
    static BufferedReader br = new BufferedReader(new InputStreamReader(System.in));  
    static BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));  
    static StringBuilder sb = new StringBuilder();  
  
    public static void main(String[] args) throws IOException {  
        int N = Integer.parseInt(br.readLine());  
        int[] ropes = new int[N];  
  
        for (int i = 0; i < N; i++) {  
            ropes[i] = Integer.parseInt(br.readLine());  
        }  
  
        Arrays.sort(ropes);  
  
        int result = Integer.MIN_VALUE;  
  
        for (int i = 0; i < N; i++) {  
            int mainRope = ropes[i];  
            result = Math.max(result, mainRope * (N - i));  
        }  
  
        sb.append(result);  
        bw.write(sb.toString());  
        bw.flush();  
        bw.close();  
        br.close();  
    }  
}
```
