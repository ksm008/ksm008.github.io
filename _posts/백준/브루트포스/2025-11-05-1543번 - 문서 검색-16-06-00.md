처음에는 indexOf 메소드로 인덱스를 찾고 substring 메소드를 통해 문자열을 계속해서 지울 생각을 했다. 하지만 substring 메소드는 호출될 때마다 새로운 String 객체를 만들어 내며 문자열을 복사한다. 때문에 시간이 더 오래 걸릴 것이다. 이를 해결하기 위해 indexOf 메소드만 사용하여 해당하는 문자열의 시작 지점을 찾고, 문서에 단어가 존재한다면 단어의 길이만큼 더해 인덱스를 늘려주며 탐색을 시키면 된다.

```java
import java.io.*;  
import java.util.*;  
  
public class Main {  
    static BufferedReader br = new BufferedReader(new InputStreamReader(System.in));  
    static BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));  
    static StringBuilder sb = new StringBuilder();  
  
    public static void main(String[] args) throws IOException {  
        String document = br.readLine();  
        String searchWord = br.readLine();  
  
        int searchWordLen = searchWord.length();  
        int idx = 0;  
        int cnt = 0;  
  
        while (true) {  
            idx = document.indexOf(searchWord, idx);  
            if (idx != -1) {  
                cnt++;  
                idx += searchWordLen;  
            } else {  
                break;  
            }  
        }  
  
        sb.append(cnt);  
        bw.write(sb.toString());  
        bw.flush();  
        bw.close();  
        br.close();  
    }  
}
```
