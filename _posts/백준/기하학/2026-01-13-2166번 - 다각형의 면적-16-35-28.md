이 문제를 풀려면 좌표만 가지고 다각형의 면적을 구하는 '신발끈 공식'을 사용해야 한다. 좌표를 나열해서 대각선으로 곱하고 빼는 방식이 마치 신발끈을 묶는 모양 같다고 해서 붙여진 이름이다.

```java
import java.io.*;
import java.util.StringTokenizer;

public class Main {
    static BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
    static BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));
    static StringBuilder sb = new StringBuilder();

    public static void main(String[] args) throws IOException {
        int N = Integer.parseInt(br.readLine());

        long[] xArray = new long[N + 1];
        long[] yArray = new long[N + 1];

        for (int i = 0; i < N; i++) {
            StringTokenizer st = new StringTokenizer(br.readLine());

            long x = Long.parseLong(st.nextToken());
            long y = Long.parseLong(st.nextToken());

            xArray[i] = x;
            yArray[i] = y;
        }

        long A = 0;
        for (int i = 0; i < N; i++) {
            A += xArray[i] * yArray[(i + 1) % N];
        }

        long B = 0;
        for (int i = 0; i < N; i++) {
            B += yArray[i] * xArray[(i + 1) % N];
        }

        double result = Math.abs((A - B)) / 2.0;

        sb.append(String.format("%.1f", result));
        bw.write(sb.toString());
        bw.flush();
        bw.close();
        br.close();
    }
}
```

알고리즘 자체는 행렬식의 사루스 전개처럼 각 대각선 진행 방향인 좌표끼리는 곱하고 전부 더하기, 그리고 뺀 값의 절댓값을 구하고 2로 나누기...
