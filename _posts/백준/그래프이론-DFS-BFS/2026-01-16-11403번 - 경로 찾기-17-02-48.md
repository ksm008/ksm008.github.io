다익스트라 알고리즘은 한 시작점에서 시작하고, 플로이드-워셜 알고리즘은 모든 시작점에서 시작한다. 둘 다 모든 지점까지의 최단 거리를 구하는 것은 같다.

플로이드-워셜 알고리즘의 핵심은 중간에 거쳐가는 정점을 기준으로 최단 거리를 갱신한다. 자신이 알고 있던 거리 비용과, 다른 경유지를 거쳐 가는 거리 비용을 비교하고 만약 경유지를 포함한 길이 더 빠르다면, 그 경로를 최단 거리로 업데이트 한다.

$$D[i][j] = \min(D[i][j], D[i][k] + D[k][j])$$

해당 문제는 단순히 경로 존재 여부만 알면 되니 i에서 k로, 동시에 k에서 j로 가는 길이 있는지만 판단하면 된다. 

태그에 플로이드-워셜이 써져있길래 뭔가 싶어서 찾아봤다. 

```java
import java.io.*;  
import java.util.StringTokenizer;  
  
public class Main {  
    static BufferedReader br = new BufferedReader(new InputStreamReader(System.in));  
    static BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));  
    static StringBuilder sb = new StringBuilder();  
  
    public static void main(String[] args) throws IOException {  
        int N = Integer.parseInt(br.readLine());  
        int[][] dist = new int[N + 1][N + 1];  
  
        for (int i = 1; i <= N; i++) {  
            StringTokenizer st = new StringTokenizer(br.readLine());  
            for (int j = 1; j <= N; j++) {  
                dist[i][j] = Integer.parseInt(st.nextToken());  
            }  
        }  
  
        for (int k = 1; k <= N; k++) {  
            for (int i = 1; i <= N; i++) {  
                for (int j = 1; j <= N; j++) {  
                    if (dist[i][k] == 1 && dist[k][j] == 1) {  
                        dist[i][j] = 1;  
                    }  
                }  
            }  
        }  
  
        for (int i = 1; i <= N; i++) {  
            for (int j = 1; j <= N; j++) {  
                sb.append(dist[i][j]).append(" ");  
            }  
            sb.append('\n');  
        }  
  
        bw.write(sb.toString());  
        bw.flush();  
        bw.close();  
        br.close();  
    }  
}
```
