원래 에라토스테네스의 체는 2,3,5 같은 자기 자신은 지우지 않지만 이 알고리즘은 자기 자신도 지운다. 그래서 기존의 에라토스테네스의 체 알고리즘을 조금만 바꿔주면 된다. 이중 반복문 안 j의 시작을 `i * i`가 아닌 그냥 i로 시작하고, 바깥 반복문의 범위를 `i <= N`으로 잡아 다른 숫자를 지우기도 전에 반복문을 탈출하는 것을 막아야 한다.

이전에 풀었던 [소수 구하기](_posts/백준/기타%20알고리즘/2025-07-21-1929번-소수-구하기-20-20-12.md) 문제와 똑같은 알고리즘이다. 반복문 변수만 다를뿐.

```java
import java.io.*;  
import java.util.*;  
  
public class Main {  
    static BufferedReader br = new BufferedReader(new InputStreamReader(System.in));  
    static BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));  
    static StringBuilder sb = new StringBuilder();  
  
    public static void main(String[] args) throws IOException {  
        StringTokenizer st = new StringTokenizer(br.readLine());  
  
        int N = Integer.parseInt(st.nextToken());  
        int K = Integer.parseInt(st.nextToken());  
  
        boolean[] isPrime = new boolean[N + 1];  
  
        for (int i = 2; i <= N; i++) {  
            isPrime[i] = true;  
        }  
  
        int cnt = 0;  
  
        outer:  
        for (int i = 2; i <= N; i++) {  
            if (isPrime[i]) {  
                for (int j = i; j <= N; j += i) {  
                    if (isPrime[j]) {  
                        isPrime[j] = false;  
                        cnt++;  
                        if (cnt == K) {  
                            sb.append(j);  
                            break outer;  
                        }  
                    }  
                }  
            }  
        }  
  
        bw.write(sb.toString());  
        bw.flush();  
        bw.close();  
        br.close();  
    }  
}
```
