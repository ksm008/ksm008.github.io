작년 7월쯤에 풀어보려고 했는데 제대로 풀지도 못했던 문제로 기억한다. 당시의 나는 '난 실버 5 문제도 제대로 못 푸는 병신이구나' 하고 자책했던것 같은데, 지금 풀어보니 이걸 왜 못풀었지 싶다. 병신이 맞기는 했나보다 싶다. 당시에는 너무나 미숙해서 이런 간단한 알고리즘 구상조차 못하는 나라 그랬을까...

알고리즘은 간단하다. 각 행/열에 2칸 이상 비어있는 구역이 몇 개나 있는지 확인하면 된다. 첨에 그냥 무지성 2칸만 있으면 되는줄 알고 2칸 찾고 break를 걸었는데 지문을 좀 꼼꼼하게 읽어봐야 했다. 뭐지 싶었네...

나중에 알고리즘 공부를 충분히 했다 싶으면 문제를 푸는데 걸린 시간도 적어보려 한다. 아직 나는 많이 부족하기 때문에 걸리는 시간 편차가 큰 편이다. 기초부터 열심히 쌓아보고, 그 후에 시간을 재도 괜찮을 것 같다. ...물론 기초는 최대한 빨리 끝내야 할 것 같지만 말이다. 아직도 너무 부족하다.

```java
import java.io.*;  
  
public class Main {  
    static BufferedReader br = new BufferedReader(new InputStreamReader(System.in));  
    static BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));  
    static StringBuilder sb = new StringBuilder();  
  
    public static void main(String[] args) throws IOException {  
        int N = Integer.parseInt(br.readLine());  
        char[][] room = new char[N][N];  
  
        int rowCnt = 0;  
        int colCnt = 0;  
  
        boolean isLinear = false;  
  
        for (int i = 0; i < N; i++) {  
            String str = br.readLine();  
            for (int j = 0; j < N; j++) {  
                room[i][j] = str.charAt(j);  
            }  
        }  
  
        for (int i = 0; i < N; i++) {  
            isLinear = false;  
            for (int j = 0; j < N - 1; j++) {  
                char ch = room[i][j];  
                char next = room[i][j + 1];  
  
                if (ch == 'X') {  
                    isLinear = false;  
                }  
  
                if (ch == '.' && ch == next && !isLinear) {  
                    isLinear = true;  
                    rowCnt++;  
                }  
            }  
        }  
  
        for (int i = 0; i < N; i++) {  
            isLinear = false;  
            for (int j = 0; j < N - 1; j++) {  
                char ch = room[j][i];  
                char next = room[j + 1][i];  
  
                if (ch == 'X') {  
                    isLinear = false;  
                }  
  
                if (ch == '.' && ch == next && !isLinear) {  
                    isLinear = true;  
                    colCnt++;  
                }  
            }  
        }  
  
        sb.append(rowCnt).append(" ").append(colCnt);  
        bw.write(sb.toString());  
        bw.flush();  
        bw.close();  
        br.close();  
    }  
}
```
