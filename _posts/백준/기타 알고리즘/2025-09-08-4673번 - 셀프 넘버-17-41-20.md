처음에는 일단 모든 셀프 넘버가 아닌 수들을 찾고 1부터 1만까지 들어있는 리스트에서 빼주는 식으로 구현했다. 그런데 이보다도 더 최적화를 할 수 있는 방법이 있다. 바로 불린 배열을 사용하여 단순 마킹만 하면 된다는 것이였다.

```java
import java.io.*;  
import java.util.ArrayList;  
import java.util.List;  
  
public class Main {  
    static BufferedReader br = new BufferedReader(new InputStreamReader(System.in));  
    static BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));  
    static StringBuilder sb = new StringBuilder();  
  
    public static void main(String[] args) throws IOException {  
        List<Integer> resultList = new ArrayList<>();  
        List<Integer> selfNumList = new ArrayList<>();  
  
        for (int i = 1; i <= 10000; i++) {  
            resultList.add(i);  
            int result = i;  
            int len = Integer.toString(i).length();  
  
            for (int j = 0; j < len; j++) {  
                result += Integer.toString(i).charAt(j) - '0';  
            }  
            selfNumList.add(result);  
        }  
  
        resultList.removeAll(selfNumList);  
  
        for (int i = 0; i < resultList.size(); i++) {  
            sb.append(resultList.get(i) + "\n");  
        }  
  
        bw.write(sb.toString());  
        bw.flush();  
        bw.close();  
        br.close();  
    }  
}
```

불린 배열로 바꾸고 난 이후는 아래와 같다.

```java
import java.io.*;  
  
public class Main {  
    static BufferedReader br = new BufferedReader(new InputStreamReader(System.in));  
    static BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));  
    static StringBuilder sb = new StringBuilder();  
  
    public static void main(String[] args) throws IOException {  
        boolean[] gen = new boolean[10001];  
  
        for (int i = 1; i <= 10000; i++) {  
            int num = i, s = i;  
            while (num > 0) {  
                s += num % 10;  
                num /= 10;  
            }  
            if (s <= 10000) gen[s] = true;  
        }  
  
        for (int i = 1; i <= 10000; i++) {  
            if (!gen[i]) sb.append(i).append('\n');  
        }  
  
        bw.write(sb.toString());  
        bw.flush();  
        bw.close();  
        br.close();  
    }  
}
```
