정수 연산을 통해 새로운 수를 계속 만들어내는 문제이다. 처음에는 정수 연산이 아니라 문자열을 일일이 파싱해가며 새로운 수를 만들었다. 

```java
import java.io.*;  
  
public class Main {  
    static BufferedReader br = new BufferedReader(new InputStreamReader(System.in));  
    static BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));  
    static StringBuilder sb = new StringBuilder();  
  
    public static void main(String[] args) throws IOException {  
        String str = br.readLine();  
        String result = str;  
        int cnt = 0;  
  
        while (true) {  
            cnt++;  
            if (result.length() == 1) {  
                result = Integer.toString(0) + result.charAt(0);  
            }  
            int temp = Integer.parseInt(result.charAt(0) + "") + Integer.parseInt(result.charAt(1) + "");  
            if (temp > 9) {  
                result = result.charAt(1) + "" + Integer.toString(temp).charAt(1);  
            } else {  
                result = result.charAt(1) + "" + temp;  
            }  
  
            if (Integer.parseInt(result) == Integer.parseInt(str)) {  
                break;  
            }  
        }  
  
        sb.append(cnt);  
        bw.write(sb.toString());  
        bw.flush();  
        bw.close();  
        br.close();  
    }  
}
```

근데 더 좋은 방법이 있다는걸 나중에 깨달았다. 일의 자리와 십의 자리를 도출해내는 더 좋은 방법은 나눗셈과 나머지 연산이다. 2자리 정수를 10으로 나누면 십의 자리가 되고, 10으로 나머지를 구하면 일의 자리가 된다. 이렇게 문자열 파싱을 없애고 정수만의 연산으로 바꾸면 연산 속도도 더 빨라질 것이다. 해당 문제에서는 큰 의미는 없겠지만...

```java
import java.io.*;  
  
public class Main {  
    static BufferedReader br = new BufferedReader(new InputStreamReader(System.in));  
    static BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));  
    static StringBuilder sb = new StringBuilder();  
  
    public static void main(String[] args) throws IOException {  
        int N = Integer.parseInt(br.readLine());  
        int result = N;  
        int cnt = 0;  
  
        while (true) {  
            cnt++;  
            int temp; int ten; int one;  
  
            if (result < 10) {  
                ten = 0;  
                one = result;  
            } else {  
                ten = result / 10;  
                one = result % 10;  
            }  
  
            temp = ten + one;  
            result = one * 10 + temp % 10;  
  
            if (result == N) {  
                break;  
            }  
        }  
  
        sb.append(cnt);  
        bw.write(sb.toString());  
        bw.flush();  
        bw.close();  
        br.close();  
    }  
}
```

오랜만에 블로그 제목 값을 한 날이였다. 평소에도 그렇긴 하지만... 오늘은 더더욱.
