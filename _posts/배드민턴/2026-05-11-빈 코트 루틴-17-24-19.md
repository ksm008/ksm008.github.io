## 단식 빈 스윙 루틴 10개
1. 우측 후위 롱 하이 클리어 → 중앙 복귀 → 좌측 전위 헤어핀    
    - 내가 **상대의 좌측 후위로** 길게 밀어친 공을 상대가 **드롭**으로 나의 좌측 전위로 짧게 놨을 때, 빠르게 달려가서 **상대의 우측 전위로** 헤어핀       
2. 좌측 후위 라운드 드롭 → 중앙 복귀 → 우측 전위 대각 푸시
    - 뒤에서 **상대의 우측 전위로** 툭 떨어뜨리는 드롭을 치고, 상대가 **언더 클리어**로 나의 우측 전위로 억지로 퍼올린 짧은 공을 전위에서 강하게 **상대의 좌측 후위로** 푸시
3. 우측 중간 사이드 수비(언더) → 중앙 복귀 → 좌측 후위 스매시    
    - 상대의 **스매시**를 **상대의 우측 후위로** 길게 걷어 올린 후, 상대가 **클리어**로 나의 좌측 후위로 친 공이 짧게 뻗지 못했을 때 **상대의 좌측 중간으로** 스매시 역공        
4. 좌측 전위 언더 클리어 → 중앙 복귀 → 우측 후위 원점프 스매시    
    - 네트 앞에서 공을 **상대의 좌측 후위로** 길게 띄워주고 상대가 **클리어**로 나의 우측 후위로 친 공이 짧았을 때, 뒤로 나가며 원점프로 **상대의 우측 후위로** 스매시        
5. 좌측 전위 헤어핀 → 우측 후위로 빠르게 백스텝 후 대각 클리어    
    - 네트 앞에서 **상대의 좌측 전위로** 헤어핀을 놓은 후 상대가 **언더 클리어**로 나의 우측 후위로 길게 공을 퍼올렸을 때, 빠르게 물러나서 대각인 **상대의 좌측 후위로** 클리어        
6. 우측 전위 런지하며 헤어핀 → 좌측 후위 홉 스텝 이동 후 스매시    
    - 오른쪽 네트 앞 **상대의 우측 전위로** 깊게 찌르고, 상대가 **언더 클리어**로 나의 좌측 후위로 길게 올린 공을 라운드 스텝으로 빠져서 **상대의 우측 중간으로** 스매시       
7. 좌측 중간 사이드 커트(드롭 수비) → 중앙 복귀 → 우측 후위 클리어    
    - 상대의 **대각 드롭**을 **상대의 우측 전위로** 네트 앞에 짧게 받아치고, 이어서 상대가 **언더 클리어**로 나의 우측 후위로 길게 넘기는 공을 쫓아가서 **상대의 좌측 후위로** 클리어        
8. 우측 후위 반스매시 → 중앙 복귀 → 중앙 전위 푸시 (직선 랠리)    
    - **상대의 우측 중간으로** 가볍고 빠른 반스매시를 때려 상대를 흔들고, 상대가 **짧은 수비성 리턴**으로 나의 중앙 전위로 어설프게 띄운 공을 바로 달려들어서 **상대의 좌측 후위로** 푸시        
9. 좌측 후위 대각 클리어 → 중앙 복귀 → 우측 중간 사이드 직선 드라이브    
    - **상대의 우측 후위로** 길게 치고, 상대가 **드라이브**로 나의 우측 중간으로 빠르게 찌르는 공을 드라이브로 **상대의 우측 후위로** 맞받아침        
10. 우측 전위 언더 → 좌측 후위 라운드 스매시 → 전위로 빠르게 대시하며 푸시
	- 공을 **상대의 좌측 후위로** 띄우고 상대가 **클리어**로 나의 좌측 후위로 올린 공을 빠져서 **상대의 우측 중간으로** 스매시를 때린 다음, 상대가 다시 **짧은 블록 수비**로 나의 중앙 전위로 받은 것을 대시해서 **상대의 좌측 후위로** 푸시로 마무리

## 복식 빈 스윙 루틴 10개
1. (전위) 숏서브 모션 → 우측 전위 푸시 → 좌측 전위 푸시    
    - 숏서브를 **상대의 중앙 전위로** 넣고 상대가 **커트 리시브**로 나의 우측 전위로 짧게 리시브하는 공을 **상대의 우측 후위로** 푸시한 뒤, 상대가 다시 **짧은 수비 반발**로 나의 좌측 전위로 뺀 공을 따라가며 **상대의 좌측 후위로** 푸시        
2. (후위) 좌측 후위 스매시 → 앞쪽으로 한 발 들어가며 직선 연속 스매시    
    - 스매시를 **상대의 좌측 중간으로** 때리고 상대의 **수비 블록**이 나의 중앙 전위로 짧게 올라왔을 때, 네트 쪽으로 전진하면서 **상대의 우측 후위로** 연속 스매시        
3. (수비) 스플릿 스텝 → 좌측 언더 수비 → 우측 언더 수비 → 좌측 언더 수비    
    - 상대의 연속 **스매시** 공격이 나의 좌측 중간, 우측 중간, 다시 좌측 중간으로 연달아 왔을 때 양옆으로 움직이며 계속 **상대의 좌측 및 우측 후위로** 길게 언더 수비        
4. (전위) 스플릿 스텝 → 우측 포핸드 드라이브 → 좌측 백핸드 드라이브    
    - 상대가 **드라이브**로 나의 우측 중간으로 빠르게 보낸 공을 **상대의 좌측 후위로** 드라이브로 치고, 연이어 상대가 **드라이브**로 나의 좌측 중간으로 찌르는 공을 백핸드 드라이브로 **상대의 우측 후위로** 맞받아침        
5. (전위) 좌측 전위 짧은 커트(차단) → 우측 전위 대각 푸시    
    - 파트너가 뒤에서 공격할 때 상대가 **드라이브 수비**로 나의 좌측 전위로 뺀 짧은 리턴을 끊어먹어 **상대의 좌측 전위로** 보내고, 상대가 다시 **언더**로 나의 우측 전위로 띄운 공을 **상대의 좌측 후위로** 푸시        
6. (후위→전위) 우측 후위 스매시 후 네트 앞으로 빠르게 대시하며 헤어핀 (로테이션)
    - 내가 스매시를 **상대의 좌측 중간으로** 때리고 파트너가 뒤로 커버하러 갈 때, 상대가 **커트 수비**로 나의 우측 전위로 짧게 받은 공을 치고 들어가며 **상대의 우측 전위로** 헤어핀
7. (후위) 우측 후위 반스매시 → 좌측 후위로 이동하며 라운드 스매시    
    - 후위에서 공격 중에 상대가 **드라이브 수비**로 나의 좌측 후위 빈 공간으로 길게 빼는 수비를 할 때, 빠르게 쫓아가며 라운드로 **상대의 우측 중간으로** 연속 스매시 공격
8. (수비→전위) 우측 사이드 언더 수비 → 앞으로 대시하며 짧게 푸시 후 전위 장악
    - 수비하다가 상대의 **드롭**이 나의 우측 전위로 짧고 뜨게 보일 때, 과감하게 치고 들어가며 **상대의 좌측 후위로** 푸시하고 전위 장악
9. (전위) 롱서브 리시브 대시 푸시 → 뒤로 빠지는 스텝 밟다가 다시 앞으로 들어가며 헤어핀
    - 상대의 **롱서브**를 나의 우측 후위에서 강하게 **상대의 좌측 중간으로** 치고 들어갔다가, 상대가 다시 **헤어핀 모션 드롭**으로 나의 우측 후위로 길게 넘기는 척하면서 나의 우측 전위로 짧게 놓은 공에 속지 않고 **상대의 좌측 전위로** 헤어핀 방어
10. (후위→전위) 좌측 후위 대각 클리어 → 네트 우측 앞으로 대각선 대시하며 전위 커버
	- 뒤에서 대각인 **상대의 우측 후위로** 길게 치고 파트너와 자리를 바꾸며, 상대가 **대각 드롭**으로 나의 우측 전위로 떨어뜨리는 공을 빠르게 **상대의 우측 전위로** 헤어핀하며 커버
