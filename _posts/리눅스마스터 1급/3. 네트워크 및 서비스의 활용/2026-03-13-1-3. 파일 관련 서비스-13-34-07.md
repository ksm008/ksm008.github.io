## 1. 삼바 사용하기
- **개요**
	- GPL 기반 자유 소프트웨어. 리눅스와 윈도우 간 디렉터리, 파일, 프린터, USB 등을 공유하는데 사용 가능.
	- SMB 프로토콜을 이용.
	- 컴퓨터 이름으로도 접속 가능. 이를 WINS라고 함.
	- CIFS는 SMB를 인터넷까지 확장한 표준 프로토콜. 커먼 인터넷 파일 시스템
- **구성 요소**
	- smbd와 nmdb로 구성되어있다.
	- **nmdb**
		- UDP 137, 138 포트를 사용. 이걸로 컴퓨터를 검색.
		- TCP 139 포트으로 컴퓨터 이름으로 접속.
		- WINS를 위한 데몬.
	- **smdb**
		- TCP 445 포트를 이용.
		- 주요 기능을 담당

### 2. 삼바 설치
`yum install samba samba-common samba-client`

## 2. NFS 서비스 사용하기
### 1. 서비스 개요
- TCP/IP를 이용하여 원격 호스트의 파일을 로컬 호스트에 있는 것처럼.
- RPC를 사용하여 관련 명령어를 전달하므로 rpcbind 데몬이 필요함.

### 2. NFS 서비스 설치와 구성
- **프로그램 설치**
	- `yum -y install rpcbind ntf-utils`
- **NFS 서버 설정**
	- **/etc/exports** 파일을 이용하여 NFS 서비스를 설정
	- **주요 옵션**
- 관련 데몬 실행
	- `systemctl start rpcbind nfs-sever`

### 3. NFS 서버 이용하기
- **연결**
	- mount 명령어로 NFs 서버에 연결함.
- **/etc/fstab을 이용한 마운트**

## 3. FTP 서비스 사용하기
- TCP/IP를 이용하여 서버와 클라이언트 간 파일 전송을 하기 위한 프로토콜
- 익명사용자도 지원하지만 보안에 취약함.
- CentOS 등 리눅스에서 기본 제공하는 vsftpd (베리 서큐어 FTP 데몬)은 가장 대표적인 프로그램. 
- 익명 사용자 디렉터리 권한이 777이면 로그인을 제한한다.
- 명령 전달을 위한 제어 포트, 전송 포트를 각각 사용한다.
	- 제어 포트는 21번 사용. 사용자 인증, 명령 등을 전달한다.
	- 동작 방식은 능동 모드와 수동 모드로 나뉜다.
		- 능동 모드는 데이터 포트로 20번을 사용
		- 수동 모드 포트 번호는 주로 1024번 이후
