## Snip의 문제점

이전에 스트리밍을 하면서 현재 스포티파이에서 듣고 있는 노래를 띄우기 위해 Snip을 사용했다. 근데 문제는 노래 제목이 전부 영어로 번역된 채 출력이 되어서 보기 좀 그렇다는 것이다. 그래서 한글로 잘 나오도록 고쳐보기로 했다.

![](assets/img/개발/기타/Snip%20한글화%20하기/Pasted%20image%2020260219140054.png)

대충 보면 이런 식이다. 

## 동작 구조
### 사용할 라이브러리
- **1. Spotify Web API for Java :** 자바와 스포티파이와 서로 통신을 위한 라이브러리이다.
- **2. Gson :** JSON 텍스트를 객체, 변수로 바꿔준다.

### 플로우
- **1. 로그인 및 인증 :** 프로그램이 일단 켜지면 토큰 파일이 있는지 확인하고 만약 없다면 브라우저를 띄운다. 스포티파이에 로그인 하도록 창을 띄우고 진행되면 로컬 서버로 인증 코드를 보내준다.
	- 로컬 서버로 인증 코드를 보내주는 이유는 스포티파이 API의 OAuth 2.0 때문이다. 스포티파이는 인증 코드를 **서버**로 보내주는데, 내 프로그램은 서버가 아니다. 그래서 잠깐 내 컴퓨터에서 받을 수 있도록 임시 웹 서버를 여는 것이다. 
- **2. 반복 :** 3초마다 스포티파이에게 듣고 있는 음악 정보를 요구한다. 그리고 그 정보를 저장한다.
