## PostgreSQL은 매우 엄격하다
MySQL은 유연하지만 포스트그레스는 매우 엄격하다. 예를 들어 작은 따옴표 (`'`)는 오직 데이터 값, 즉 문자열, 날짜를 감쌀 때만 사용한다. 큰 따옴표 (`"`)는 테이블 이름, 컬럼 이름 등 식별자를 감쌀때만 사용한다. MySQL에서의 백틱  ( \` ) 역할을 한다.

또한 따옴표 없이 쓴 이름은 무조건 소문자로 변환하여 저장한다.

```sql
create table TestTable ( ... );

-- PostgreSQL에서는 테이블 이름을 testtable로 저장함
```

## CMD에서 유저 및 테이블 생성

### 1. PostgreSQL 관리자 접속

```bash
psql -U postgres
```

### 2. 유저, 테이블 생성 및 권한 부여

```sql
create user spring with password '1234';

create database simple_crud owner spring;

-- db, 유저 변경
\c simple_crud spring 

grant all on schema public to spring;
```
